全局安装命令
	Mac|Linux 
		将composer.phar 拷贝到/usr/local/bin下 cp composer.phar /usr/local/bin/composer 
		给composer赋权限 sudo chmod -R 755 composer
	Windows
		将composer.phar 拷贝到php.exe同级目录下
		新建文件composer.bat ,写入 @php "%~dp0composer.phar" %* ，
		在任意目录下都可执行composer命令

中国镜像配置查看

composer config -gl

composer config -g repo.packagist composer https://packagist.phpcomposer.com


主要命令
 search
 show
 require
 install
 update

例子：查看monolog
 composer search monolog
选择版本信息
 composer show --all monolog/monolog
下载安装
在 conposer.josn中 写入"require": {"monolog/monolog":"1.21.*"}
 composer install
更新
删除"monolog/monolog":"1.21.*",
 composer update
